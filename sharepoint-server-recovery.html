Notable features of SharePoint Server Recovery software

SharePoint Recovery Tool

There are two recovery modes for handling different level of corruption. For e.g. for minor corruption, Standard mode of recovery is capable enough to recover data from corrupt database. In case of severe corruption, Advanced mode of recovery can be useful

SQL Database Recovery

Another recovery option is "Complete Database Recovery". You should choose this option if you want to recover the entire data (everything) from the SharePoint Server database. SQL Recovery software can recover BLOB, image or large page data from corrupt SQL database.

Smart & Quick Scanning

Advanced Sharepoint Recovery tool is designed to quickly scan the corrupt database in a smart way. Being a non-destructive tool, it causes no alteration or changes during the scanning process

Two Recovery Modes

For SharePoint database recovery, there are two recovery modes: Standard for minor corruption and Advanced for severe corruption. First, choose the standard mode, and then advanced mode

Enable Collation Option

Since SQL Server saves database files with a particular collation type, the Collation option in this SharePoint recovery tool will be helpful for you. You can either choose the auto-detect option or you can define a collation type manually.

Deleted Records Preview

If you want to see the preview of the deleted records, there are two options to choose before you can scan the corrupt SharePoint database:"Show deleted records" and "Show advanced deleted records".

Smart Database Preview

After a successful scanning of the corrupt database, you can check the preview of your data which you can recover from the database. The data preview is shown in a tree-structure mode.

Log File Generation

Software automatically creates a log file in text format at the location you've defined earlier. The log file stores the complete information about SharePoint Server recovery.

Interactive GUI

SharePoint Server Recovery software contains an easy and interactive graphical user interface that requires no prior technical expertise. Even a novice user can use this software with ease.
